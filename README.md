# Sistema de Gestión de Justificaciones de Inasistencias

## Descripción

Sistema web para la gestión de justificaciones de inasistencias estudiantiles del Instituto Valle Grande de Cañete.

## Características

- Dashboard con estadísticas
- Gestión de estudiantes
- Revisión de justificaciones
- Reportes y análisis

## Tecnologías

- React
- Tailwind CSS
- Vite

## Instalación

```bash
npm install
npm run dev
```
